# Python Pizzeria 🍕

A console-based pizza ordering system built in Python that simulates 
a real restaurant ordering experience — from menu browsing to order 
customization and receipt generation.

## How It Works
- Browse a menu of available pizzas, sizes, and toppings
- Customize your order with add-ons and quantity selection
- Review your full order with itemized pricing
- Receive a generated receipt with order total

## Features
- Interactive menu-driven console interface
- Pizza customization — size, toppings, quantity
- Dynamic price calculation based on selections
- Order summary and receipt generation
- Input validation to handle invalid menu choices

## Concepts Demonstrated
- Functions and modular code organization in Python
- Dictionary and list data structures for menu management
- Control flow with loops and conditionals
- User input handling and validation
- String formatting for clean receipt display

## How to Run
```bash
# Make sure Python 3 is installed, then:
python "PYTHON PIZZERIA.py"
```
